test\_version module
====================

Overview
--------

This script enables you to print the versions of the necessary libs:
   * "torch"
   * "torchvision"
   * "torchaudio"
   * "torchtext"
   * "datasets"
   * "tokenizers"
   * "torchmetrics"
   * "tensorboard"
   * "altair"
   * "wandb"
